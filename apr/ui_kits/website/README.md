# Saint Laurent — Website UI kit

Interactive recreation of the Saint Laurent marketing site, reflecting the final brand reveal direction (not the `codex v1` experiment, which used placeholder Oswald/Space Mono and CSS-drawn food shapes).

## Surfaces demonstrated
- **Nav** — sticky cream bar, horizontal wordmark, red CTA
- **Hero** — forest panel, grid overlay, tilted fleur logomark, giant Bebas Neue "MONTREAL SOUL, ALBERTA SOIL." with red/cream checker footer
- **Menu** — cream section, Bebas Neue item names in Québec blue, red prices, dotted rules, section rules in red
- **Story** — forest split-panel with interior photo, tilted red circle badge, stats row
- **Gallery** — three editorial photos; center shot has the "JUICY AF" poster overlay
- **Visit** — cream-warm panel, left info column, right reservation form with validation-lite interaction
- **Footer** — char dark footer, checker at top, wordmark + columns + civic-style meta line
